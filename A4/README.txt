Instructions:
- To run the program, use python RuleMining.py
- Ensure that the data set 'Play_Tennis_Data_Set.csv' is located within the same directory as the python script

Description:
RuleMining.py is an implementation of the apriori algorithm. 
The user is asked to enter a minimum support and a minimum confidence in order to generate the corresponding association rules.
The results of the association rule computations are stored and accessible in the text file Rules.txt
